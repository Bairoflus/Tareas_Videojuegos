Comandos de navegación

  • pwd: muestra el directorio actual.
  • ls: lista los archivos y carpetas del directorio actual.
  • cd : cambia al directorio especificado.
    • cd ..: sube un nivel en la jerarquía de carpetas.
    • cd ~: va a mi directorio personal (home).
  • mkdir : crea una nueva carpeta.
  • rmdir : elimina una carpeta vacía.
  • rm -rf : elimina una carpeta y todo su contenido de manera forzada.

Manuales de ayuda

  • man : muestra el manual del comando.
  • --help: muestra una ayuda rápida sobre cómo usarlo.

Comandos de gestión de archivos

  • touch : crea un archivo vacío con ese nombre.
  • cp : copia el archivo de origen al destino.
  • mv : mueve o renombra el archivo.
  • rm : elimina un archivo.
