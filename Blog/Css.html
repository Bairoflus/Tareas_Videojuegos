CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de documentos HTML. Permite aplicar estilos como colores, márgenes, tipografías, y más.

Conceptos clave

  • Selector: Indica a qué elementos HTML se les aplicará un estilo.
  • Propiedad: Define el tipo de estilo que se qu]iere aplicar (por ejemplo: color, margin).
  • Valor: Especifica cómo se aplicará esa propiedad (por ejemplo: red, 20px).

Tipos de selectores

  1. Por tipo: Selecciona elementos por su nombre (por ejemplo, `p`, `h1`).
  2. Por clase: Se usa el punto (`.`) para seleccionar elementos con cierta clase. Ejemplo: `.menu`.
  3. Por ID: Se usa el símbolo `#` para seleccionar un elemento único. Ejemplo: `#principal`.

Propiedades comunes en CSS

  • font-family: Cambia la fuente. Ej: `font-family: Arial, sans-serif;`
  • font-style: Aplica cursiva. Ej: `font-style: italic;`
  • color: Cambia el color del texto. Ejemplo: `color: blue;`
  • font-weight: Aplica grosor al texto. Ej: `font-weight: bold;`
  • background-color: Establece un color de fondo. Ej: `background-color: #f2f2f2;`
  • background-image: Aplica una imagen de fondo. Ej: `background-image: url('fondo.png');`
  • width / height: Define el ancho y alto de un elemento. Ej: `width: 300px; height: 150px;`
  • border: Aplica un borde completo. Ej: `border: 2px solid black;`
  • border-left / border-right / border-top / border-bottom: Aplica bordes a lados específicos.
  • border-radius: Redondea las esquinas. Ej: `border-radius: 10px;`
  • padding: Espacio interno entre el contenido y el borde del elemento. Ej: `padding: 20px;`
  • margin: Espacio externo entre el elemento y los demás. Ej: `margin: 10px;`
  • text-align: Alinea el texto. Ej: `text-align: center;`
  • display: define como se muestra un elemento en la pagina. Ej: `display: flex;`
  • justify-content: Define como se distribuyen los elementos a lo largo del eje horizontal. Ej: `justify-content: center;`
  • align-items: Define como se distribuyen los elementos a lo largo del eje vertical. Ej: `align-items: center;`
  • flex-direction: Define la dirección de los elementos dentro de un contenedor flex. Ej: `flex-direction: row;`
